# Payment System (Golang)

## Описание

Простой платёжный сервис на Go, реализующий переводы между пользователями.

Проект демонстрирует:
- работу со структурами
- обработку ошибок
- конкурентность (goroutines, channels)
- защиту данных (mutex)

---

# Версии проекта 

## v1 - базовая структура

Что добавлено: 
- структура User
- методы Withdraw, Deposit

Почему: 
- создать фундамент системы

## v2 - обработка транзакция

Что добавлено:
- структуры Transaction, PaymentSystem
- методы AddUser, AddTransaction, ProcessingTransaction
- обработка ошибок

Почему: 
- реализовать бизнес-логику переводов


## v3 - конкурентная обработка

Что добавлено: 
- горутины
- канал как очередь задач
- worker pool
- sync.WaitGroup

Почему:
- увеличить производительность за счет конкурентной обработки

## v4 - защита данных(операции перевода не атомарны)

Что добавлено:
- sync.Mutex в User
- защита Deposit и Withdraw

Почему:
- избежать data race при одновременном доступе к Balance
